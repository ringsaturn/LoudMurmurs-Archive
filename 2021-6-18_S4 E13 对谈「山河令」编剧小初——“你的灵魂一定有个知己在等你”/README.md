<p>【假装听过】</p>
<p>小初解释文艺作品的内核——你把自己的灵魂develop到一个完整的程度，一定会有一个知己在千万人中等待着你，有可能因为阴差阳错，你们终生无法相遇，但是通过网络平台也好，通过文艺作品的连接也好，你会看到千千万万个跟你的灵魂碎片，能够产生共振和共鸣的人存在，你就不再孤独，这是文艺作品存在的最美好的意义之一。</p>
