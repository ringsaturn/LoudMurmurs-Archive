<p>【聊了什么】</p>
<p>HBO新出的迷你剧 Mare of Easttown《东城梦魇》讲述了宾州蓝领白人社区发生的故事。这期节目里，Izzy，Diaodiao，Ina还有嘉宾Alex Lawyer一起聊了聊这部剧的牛逼之处（角色，剧情，设定）以及谋杀案剧情背后的社会议题，比如被时代扔下的铁锈带小镇居民所经历的社群衰败和
鸦片类药物上瘾对社群的影响。</p>
